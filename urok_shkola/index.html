Мова HTML

Історія мови

Опис мови HTML вперше був запропонованний у 1991 році британським вченим Тімом Бренес-Лі (нар. у 1955 р.) У документі з назвою HTML Tags містився опис перших 20 тегів для розмітки документів

У 1994 році Тім Бренес-Лі заснував Консоціум Всесвітньої павутини WЗC (англ. World Wibe Web Condortium - консоціум всесвітньої павутини), який очолює до сьогодні. Консоціум займається розробкою і впровадженням стандартів для Інтернету, у тому числі затверджує специфікацію мови HTML. У 2014 році був рекомендований стандарт п'ятої версії мови - HTML5.

Основні поняття

Teг

- це не команда браузеру, яка вказує, яким чином повинен бути відображений той чи інший фрагмент коду.

Контент

- це тектс та нетекстові об'єкти (графічні зображення, анімація, відео та ін.), що відображаються на веб-сторінці та призначені для перегляду відвідувачами веб-сторінкию
